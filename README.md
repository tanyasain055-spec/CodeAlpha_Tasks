Steam Top Sellers Analysis
Project Overview
This project was completed as part of the CodeAlpha Data Analytics Internship.

The project focuses on collecting and analyzing Steam Top Sellers data using Python. Data was scraped from the Steam Store and transformed into a structured dataset for analysis and visualization.

Objectives
Perform web scraping using BeautifulSoup
Create a custom dataset from Steam
Conduct exploratory data analysis (EDA)
Build visualizations to uncover insights
Dataset Features
Game Name
Price
Release Date
Clean Price
Tools Used
Python
BeautifulSoup
Pandas
Matplotlib
Tasks Completed
Task 1: Web Scraping
Collected Steam Top Sellers data including game names, prices, and release dates.

Task 2: Exploratory Data Analysis
Analyzed pricing trends, free vs paid games, and identified the most expensive games.

Task 3: Data Visualization
Created charts for:

Free vs Paid Games
Top 10 Most Expensive Games
Release Year Distribution
Price Distribution
Key Insights
Several top-selling games are free-to-play.
Game prices vary significantly across the dataset.
Recent releases dominate the Top Sellers list.
A small number of games account for the highest prices.
Conclusion
This project demonstrates the complete data analytics workflow including data collection, cleaning, analysis, and visualization.
